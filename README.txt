Description

  The LibDeskAction C++ library allows to interact automatically in an 
easy way with one window or with the desktop in the windows operating 
systems, by reading out the contents of the window itself as one RGB 
buffer and, basing on its contents, the library allows to issue synthetic 
mouse or keyboard interaction events. This permits one straightforward 
simple way to create desktop interaction bots.
